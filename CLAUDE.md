# CLAUDE.md — ponzu-oyatsu-claude

## この repo について
「ぽんず（本物）のおやつ代ゲーム」。愛犬ぽんずの「できること」達成を記録し、喜びポイントで庭を強化していく成長ゲーム（PWA）。単一 index.html（VanillaJS）＋ sw.js。

## 必ず守ること
- 数値調整パラメータ（1回あたりの喜びポイント・施設強化コスト等）は index.html 内の `CONFIG` オブジェクトに集約する。
- 課金導線のリンク先は `SUPPORT_LINK_URL` 定数1箇所のみを差し替える（決済処理・外部連携は未実装のまま）。
- 単一HTML構成を維持する（外部フレームワーク不使用、VanillaJS）。
- セーブは localStorage のみ（キー: `ponzu_oyatsu_v1`）。外部保存・サーバーは使わない。
- リポジトリは private 運用。

## ここに書かないこと
- 実装フェーズの進捗・仕様変更の詳細はここに書かない（必要ならノア側の企画ファイルを参照）。
